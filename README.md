# Videogame Club Stats

Benvenuti nella repository del sito riguardante le statistiche del videogame club!

Il sito attualmente mostra 4 statistiche inerenti la partecipazione al videogame club, ovvero:


- Presenze Totali
- Presenze Settimanli
- Titoli giocati
- Generi giocati
